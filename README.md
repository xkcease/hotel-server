# 酒店预订系统-Node.js服务端
Node.js + Express

该仓库为酒店预订系统的Node.js服务端，其前台微信小程序和Vue后台如下：
> [微信小程序前台](https://github.com/xkcease/hotel-weapp)  

> [Vue后台](https://github.com/xkcease/hotel-management)

## 技术栈
- Node.js
- Express
- Mysql
- ES6

## 安装与使用
```shell
# 安装依赖
npm install

# 运行
node app.js
		
```
